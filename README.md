# Ismism-Machine

## 交互式艺术与哲学研究展示网站

这是一个基于Next.js, TypeScript和Tailwind CSS开发的交互式网站，旨在以创新和艺术化的方式展示艺术与哲学研究内容。

### 项目特点

- **唱片封面式浏览体验** - 内容以唱片封面形式呈现，用户可水平滚动浏览
- **时间轴可视化** - 支持拖拽和缩放的D3.js时间轴
- **模块化便当布局** - 灵活的内容展示布局
- **动画与交互效果** - 使用GSAP实现流畅的页面过渡和视觉效果

### 技术栈

- **前端框架**: Next.js (React)
- **样式系统**: Tailwind CSS
- **动画库**: GSAP, Lenis
- **数据可视化**: D3.js
- **3D效果**: Three.js (按需使用)

### 设计理念

项目灵感来源于Cologne on Pop GmbH网站的设计风格，结合现代设计元素与怀旧感，创造一种独特的浏览体验。详细的设计参考文档可在`docs/design-references/cologne-on-pop/`目录下查看。
